# 二、计算流程（你这张表的“算法”）

我给你还原成一个标准公式链：

------

## Step 1️⃣ 计算每日总食量

```
每日食量(g) = 体重(kg) × 系数
```

```
≈ 70 g / kg
```

## Step 2️⃣ 多天总量

```
总猫饭量(g) = 所有猫每日食量 × 天数
```

你现在是：

```
210g × 15天 = 3150g
```

------

## Step 3️⃣ 考虑出成率（关键）

```
原料总量 = 总猫饭量 ÷ 出成率
```

你设定：

```
出成率 = 87%
```

所以：

```
3150 ÷ 0.87 ≈ 3621g
```

👉 ✔ 这就是“需要采购的原料总重量”

------

## Step 4️⃣ 各原料分配

```
某原料重量 = 原料总量 × 对应比例
```

例如：

```
鸡腿 = 3621 × 26% ≈ 941g
鸡心 = 3621 × 15% ≈ 543g
……
```

------

## Step 5️⃣ 添加剂计算

```
添加剂量 = 总猫饭量 × 添加比例
```

注意：这里用的是 **成品猫饭量（3150g）**

```

```

- 碘：


# 四、约束条件（重点）

## 1️⃣ NRC最低需求约束

对每个营养素 (i)：

[
\sum_j a_{ij} x_j \geq b_i
]

其中：

- (b_i)：NRC推荐摄入量

------

## 2️⃣ 上限约束（防过量）

例如：

### 维生素A：

[
\sum_j a_{VA,j} x_j \leq U_{VA}
]

------

### 钙磷比：

这是关键非线性约束：

[
1.1 \leq \frac{Ca}{P} \leq 1.5
]

可转化为线性：

[
Ca - 1.1P \geq 0
]
[
Ca - 1.5P \leq 0
]

------

## 3️⃣ 牛磺酸约束

[
\sum_j a_{tau,j} x_j + x_{supp} \geq b_{tau}
]

👉 (x_{supp})：额外补充量

------

## 4️⃣ 食材比例约束（工程约束）

为了避免“数学最优但猫不吃”：

例如：

[
0.2 \leq \frac{x_{\text{肌肉}}}{\sum x} \leq 0.7
]

[
x_{\text{肝}} \leq 0.1 \sum x
]

[
x_{\text{鱼}} \leq 0.05 \sum x
]

------

## 5️⃣ 总量约束

[
\sum_j x_j = R
]

👉 R = 你算出的总猫饭量

------

# 五、目标函数设计（关键区别）

## ✔ 方法1：最小偏离（推荐你）

[
\min \sum_i w_i (\sum_j a_{ij} x_j - b_i)^2
]

👉 优点：

- 不会极端解
- 更稳定

## ✔ 方法2：最小成本

[
\min \sum_j c_j x_j
]

👉 适合实际采购优化

------

## ✔ 方法3：混合（工程最优）

[
\min \lambda_1 \text{营养偏差} + \lambda_2 \text{成本}
]

# 五、cvxpy代码（可直接跑）

```python
import cvxpy as cp
import numpy as np

# 变量
x = cp.Variable(8, nonneg=True)
x_ca = cp.Variable(nonneg=True)
x_tau = cp.Variable(nonneg=True)

R = 3470

# 营养矩阵（按g单位）
A_protein = np.array([0.18,0.16,0.17,0.13,0.17,0.20,0.20,0.20])
A_phos = np.array([1.8,2.1,2.0,1.9,3.3,3.5,2.0,2.3])
A_ca = np.array([0.1,0.1,0.1,0.5,0.1,0.1,0.1,0.2])
A_tau = np.array([0.5,1.2,1.5,0.2,0.3,0.3,0.4,0.8])

constraints = [
    cp.sum(x) == R,
    
    A_protein @ x >= 528,
    A_phos @ x >= 4500,
    
    A_ca @ x + x_ca >= 5500,
    
    (A_ca @ x + x_ca) >= 1.1*(A_phos @ x),
    (A_ca @ x + x_ca) <= 1.5*(A_phos @ x),
    
    A_tau @ x + x_tau >= 3300,
    
    x[4] + x[5] <= 0.07 * R,
    x[7] <= 0.05 * R,
]

# 目标：最均衡
objective = cp.Minimize(cp.sum_squares(x - R/8))

prob = cp.Problem(objective, constraints)
prob.solve()

print(x.value, x_ca.value, x_tau.value)
```
