## 先说结论

这套猫饭现在的原理已经明确统一成下面 5 条：

1. **鱼固定用鲅鱼，不再用大西洋鲭鱼口径。**
2. **鱼比例固定 4%，肝脏总量固定 7%，鸡皮固定 7%。**
3. **允许鸡腿与鸭胸互换，形成 A / B / C 三个现行版本。**
4. **3 个版本共用同一套补充剂，并且都额外补 1 粒 VD3 1000IU。**
5. **当前 29 项 NRC 复核都已通过；原理、脚本、执行文档必须使用同一套鲅鱼数据。**

## 一、旧版到底错在哪

之前最容易让人混乱的地方，不是“大方向错”，而是**旧定量口径残留**：

| 项目 | 旧残留口径 | 现行正确口径 |
|---|---|---|
| 鱼种 | 大西洋鲭鱼 | **鲅鱼（Spanish mackerel）** |
| 皮脂比例 | 9% | **7%** |
| 配方结构 | 单一旧版 825g 鸡腿 + 285g 鸡皮 | **A / B / C 三版本** |
| 维生素D逻辑 | 主要靠旧鱼种天然 VD | **鲅鱼天然 VD 余量偏薄，统一补 VD3** |
| 维生素E规格 | 400IU × 2 粒 | **100IU × 8 粒** |
| 锌 / 铁规格 | 15mg×2、18mg×2 | **30mg×1、36mg×1** |
| 批次总热量 | 约 4954 kcal | **A/B/C 分别为 4724 / 4820 / 4883 kcal** |
| 能量密度 | 1.737 kcal/g | **1.657 / 1.691 / 1.706 kcal/g** |

这也是为什么现在必须把原理文档改成现行体系：

> **原理可以保留，但定量必须跟现行执行版一致。**

## 二、现行配方原理

### 1️⃣ 三版本的共同框架

| 类别 | A版：7%鸡皮基础版 | B版：7%鸡皮+鸭胸版 | C版：鸭胸强化版 |
|---|---:|---:|---:|
| 鸡腿肉（去皮去骨） | 28% | 9% | 0% |
| 鸭胸肉（去皮） | 0% | 19% | 28% |
| 猪瘦肉/里脊 | 19% | 19% | 19% |
| 鸡心 | 13% | 13% | 13% |
| 牛心 | 14% | 14% | 14% |
| 鸡蛋（全蛋） | 8% | 8% | 8% |
| 鸡肝 | 4% | 4% | 4% |
| 牛肝 | 3% | 3% | 3% |
| 鲅鱼 | 4% | 4% | 4% |
| 鸡皮 | 7% | 7% | 7% |

### 2️⃣ 为什么这样定

| 设计点 | 当前原则 | 原因 |
|---|---|---|
| 主鱼种 | 鲅鱼固定 4% | 提供 EPA/DHA，但避免鱼过高带来的挑食与重金属暴露 |
| 肝脏总量 | 固定 7% | 提供 VA、铜、B12，同时避免 VA 过量 |
| 皮脂比例 | 固定 7% | 兼顾能量密度、花生四烯酸与亚油酸，不再沿用旧 9% |
| 鸡腿 / 鸭胸切换 | 只在 A/B/C 内部切换 | 让库存处理更灵活，但不改变整体营养结构 |
| 鸭胸数据口径 | 牛磺酸与花生四烯酸按 0 做保守计算 | 即使按保守值，当前 29/29 仍通过 |
| 钙来源 | 18g 碳酸钙 | 精确拉回 Ca:P 到安全区间 |
| 维生素D | 每批统一补 1 粒 VD3 1000IU | 鲅鱼天然 VD 明显低于旧大西洋鲭鱼口径 |

### 为什么现在必须额外补 VD3

`verify_spanish_mackerel.py` 的现行结论是：

| 版本 | 不补 VD3 时天然 VD | 补 1 粒 VD3 后 |
|---|---:|---:|
| A | 86.1 IU/1000kcal | 297.8 IU/1000kcal |
| B | 78.2 IU/1000kcal | 285.6 IU/1000kcal |
| C | 74.2 IU/1000kcal | 279.0 IU/1000kcal |

> NRC 下限是 **70 IU/1000kcal**。  
> 也就是说：**不补时虽然还能过线，但余量太薄；补 1 粒后才进入更稳的执行区间。**

## 七、制作与执行原则

### 采购 / 制作时最重要的规则

1. **鸭胸一律按去皮后重量计算。**
2. **鱼保持 4%，不要擅自增加。**
3. **肝脏总量保持 7%，不要上调。**
4. **鸡皮保持 7%，不要再沿用旧 9% 口径。**
5. **碳酸钙、牛磺酸、碘、锰、铁、VD3 不能删。**
6. **补充剂必须在 40°C 以下加入。**
7. **血水和汤汁全部保留。**
8. **冷冻建议不超过 2 个月。**

### 什么时候必须重算

只要出现下面任意一条，就不要再照旧文档直接做：

- 鸡皮明显低于 7% 或高于 7%
- 鲅鱼不是 127g / 4% 左右这个量级
- 鸭胸继续上调到 1000g 以上
- 想把鱼油、VD3、碳酸钙删掉或换规格
- 想把肝脏、心脏、蛋比例做明显改动

### 鸡皮临时买不够时的应急规则

这个情况在现实里很常见，当前现行口径可以这样处理：

- **鸡皮不足时，缺多少克，就用去皮鸡腿肉补多少克。**
- **可以低于 7% 鸡皮做这一锅，但不要顺手改鱼、肝、补充剂和 VD3。**
- **即使完全不用鸡皮，这锅也不是立刻变成“不能吃”；核心变化是热量下降，而不是 29 项营养马上失效。**
- **代价是能量密度下降，所以后面往往要把每日猫饭量稍微加一点。**
- **鸭皮不建议直接顶替鸡皮，除非后续重新计算。**

简单说：

> **鸡皮买不够时，优先方案不是停做，而是“缺多少，用去皮鸡腿肉补多少”，然后在喂食端根据体重和食量微调。**

## 八、幼猫与长期执行说明

### 幼猫C为什么仍然能用这套结构

这套方案本质上是：

- 猫饭负责提供高水分、高动物蛋白和脂溶性营养底盘
- 干粮补足一部分幼猫阶段更高的总钙磷需求
- 幼猫分到的猫饭量最多（80g）

所以当前做法不是“全湿粮幼猫配方”，而是：

> **成猫/幼猫共用同一锅猫饭，再靠分量和全阶段干粮把幼猫额外需求补齐。**

如果以后要改成**完全不喂干粮**，那就必须按幼猫标准重新整锅计算，不能直接沿用当前口径。

## 九、轮换与禁忌原则

### 可以轮换，但必须重算的食材

| 当前食材 | 可替换思路 | 注意 |
|---|---|---|
| 鸡腿肉 | 鸭腿、火鸡腿 | 替换后需重算脂肪与矿物质 |
| 猪瘦肉 | 兔肉、其他低脂红肉 | 太瘦时需注意能量下降 |
| 鸡心 | 鸭心 | 通常问题不大，但仍建议重算 |
| 牛心 | 羊心 | 同上 |
| 鸡肝 | 鸭肝 | 仍需盯 VA |
| 牛肝 | 羊肝 | 仍需盯 VA 和铜 |
| 鲅鱼 | 沙丁鱼、秋刀鱼等海鱼 | **必须整表重算，尤其 VD 与 EPA+DHA** |

### 明确不建议使用

| 食材 | 原因 |
|---|---|
| 金枪鱼、旗鱼 | 汞高、挑食风险高 |
| 生淡水鱼 | 寄生虫风险 |
| 生虾、生高硫胺素酶鱼 | 破坏 VB1 |
| 洋葱、大蒜 | 对猫有毒 |
| 葡萄、葡萄干 | 对猫有毒 |
| 鱼肝油 | 当前体系已不需要，容易把 VA / VD 推高 |
