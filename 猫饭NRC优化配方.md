## 📋 基本信息

### 三只猫的情况

| 猫 | 年龄 | 体重 | 阶段 | 建议每日热量 |
|---|---|---|---|---|
| 公猫A | 1岁8个月 | 5.8 kg（11.6斤） | 成猫（偏胖） | ≈220–250 kcal |
| 母猫B | 1岁8个月 | 4.5 kg（9斤） | 成猫（正常） | ≈200–220 kcal |
| 幼猫C | 5个月 | 2.5 kg（5斤） | 幼猫（生长期） | ≈280–300 kcal |

> **RER**（静息能量需求）= 70 × 体重(kg)^0.75
>
> - 公猫A：RER ≈ 70 × 5.8^0.75 ≈ **262 kcal**；偏胖减重阶段可按约 **0.85–0.95 × RER**，即 **≈220–250 kcal/天**
> - 母猫B：RER ≈ 70 × 4.5^0.75 ≈ **216 kcal**；正常维持可按约 **0.95–1.05 × RER**，即 **≈200–220 kcal/天**
> - 幼猫C：RER ≈ 70 × 2.5^0.75 ≈ **139 kcal**；5月龄生长期可按约 **2.0–2.2 × RER**，即 **≈280–300 kcal/天**

👉 **当前这套分配实际提供约 730 kcal/天，与三只猫现阶段需求基本匹配。**

> **为什么加回皮脂？** 原配方去皮会导致脂肪严重不足。NRC推荐成猫至少 **22.5g 脂肪/1000kcal ME**，

> 如果公猫A需要减肥，可以减少它的皮脂份额，但不能完全去除。

> ⚠️ **这是原始两个配方最大的短板。** 没有完善的添加剂，猫饭再好也会营养缺乏。
> 以下添加剂按 **当前精确配方约 2,851g 成品猫饭** 计算。

### NRC推荐摄入量 vs 本配方对照（以1000 kcal为基准）

本配方约含 **≈1.737 kcal/g**（湿基，厚泥状），2,851g 成品 ≈ **4,954 kcal**（15天），即 **≈330 kcal/天**（猫饭部分）。

### 🍳 制作步骤

#### Step 1：食材处理（约30分钟）

1. **鸡腿肉**：去皮去骨（皮另外保留285g），切成2-3cm块
2. **猪瘦肉/里脊**：切成2-3cm块
3. **鸡心/牛心**：对半切开，清洗干净
4. **鸡蛋**：水煮8分钟，去壳
5. **鸡肝/牛肝**：切小块，⚠️ **血水全部保留**
6. **鱼**：去鳞去内脏，切段
7. **鸡皮**：切小块

#### Step 2：烹饪（约40分钟）

> ⚠️ **所有汤汁必须全部保留！** 牛磺酸等水溶性营养大量溶于汤中。

1. **大锅加水500ml**，先下鸡腿肉、猪瘦肉、牛心，中火煮至刚刚断生（内部微粉）
2. 加入**鸡心**，继续煮5分钟
3. 加入**肝脏**（鸡肝+牛肝），煮10分钟确保熟透
4. **鱼**单独蒸10分钟
5. **鸡蛋**已水煮，去壳备用
6. **鸡皮**烤箱180°C烤15分钟或煎熟（保留油脂）

> 📝 **为什么不用炖/boil太久？** 过度烹饪会严重破坏牛磺酸、B族维生素。
> 刚断生即可，然后靠打碎来改善质地。

#### Step 3：打碎混合（约20分钟）

1. 所有熟肉、蛋、鸡皮 + **全部汤汁** 放入料理机/绞肉机
2. 打成**细泥状**（幼猫容易吃）或**粗粒状**（成猫促进咀嚼）
3. 如果太稠，可用煮肉汤调节到合适稠度

#### Step 4：添加补充剂（最关键！）

> ⚠️ 必须等猫饭**冷却到40°C以下**再添加补充剂！高温会破坏维生素。

1. 将碳酸钙粉 18g 均匀撒入，搅拌
2. 牛磺酸粉 3g 的 **一半（1.5g）** 加入，搅拌均匀
3. 鱼油胶囊 10粒 → 剪开挤入，搅拌
4. 维生素E 2粒 → 打开胶囊倒入，搅拌
5. 维生素B族 2片 → 碾碎成粉加入，搅拌
6. 碘粒 7粒 → 碾碎加入，搅拌
7. 碘盐 3g → 直接加入，搅拌
8. 锌片 2粒 → 碾碎加入，搅拌
9. 螯合铁 2粒 → 碾碎加入，搅拌
10. Source Naturals锰片 1粒 → 碾碎加入，搅拌

> **剩下的1.5g牛磺酸粉** → 分成15份（每份0.1g），每次解冻喂食时现加。
> 这样可以最大程度减少牛磺酸在冷冻过程中的降解。

#### Step 5：分装冷冻

1. 每份约190g（一天量），分成15份
2. 使用密封容器或密封袋
3. 当天吃的放冷藏，其余立即冷冻
4. **冷冻保存不超过2个月**

#### Step 6：每日喂食

1. 前一天晚上将次日份从冷冻移至冷藏自然解冻
2. 喂食前微波10-15秒加热至**接近体温**（约38°C）
3. 加入当日份牛磺酸（约0.1g）
4. 按分量分给三只猫

## 七、重要注意事项

### ❗ 安全红线

1. **肝脏总量 ≤ 7%**：超过可能导致维生素A中毒（急性/慢性）
2. **鱼类 ≤ 5%**：重金属风险、硫胺素酶（生鱼破坏VB1）、挑食成瘾
3. **不加生蛋白**：生蛋白含抗生物素蛋白（卵白素），会阻碍生物素吸收。煮熟后无此问题
4. **不用鱼肝油**：肝脏已经提供充足VA和VD，鱼肝油会导致过量
5. **不用骨粉代替碳酸钙**：除非你能精确测量含钙量
6. **不加蔬菜/谷物**：猫是专性食肉动物，消化碳水化合物的能力有限（肝糖激酶活性低、唾液淀粉酶缺失），本配方以纯肉为基底无需额外植物性成分

### ❗ 操作要点

1. **血水/肝血绝不丢弃** → 含铁、牛磺酸、水溶性维生素
2. **煮肉汤全部保留** → 牛磺酸等溶于汤中
3. **补充剂冷却后加** → 高温破坏维生素B/E/牛磺酸
4. **牛磺酸分两次加** → 制作时一半 + 喂食时一半（减少降解）
5. **碳酸钙必须用秤** → 不要用勺子估计，钙磷比是生死线
6. **冷冻不超过2个月** → 营养素随时间降解

### ❗ 每只猫的特别注意

| 猫 | 注意事项 |
|---|---|
| 公猫A（偏胖） | 控制干粮量40g，猫饭中减少鸡皮份额；监测体重，目标每月减0.1-0.2kg |
| 母猫B（正常） | 维持当前方案即可 |
| 幼猫C（5月龄） | 多分配猫饭（80g）；确保爱肯拿为全阶段配方（已满足）；6月龄后可逐渐调整至成猫比例 |

## 附录：cvxpy优化代码（可运行验证）

```python
import cvxpy as cp
import numpy as np

# 食材变量（g）：鸡腿、猪肉、鸡心、牛心、蛋、鸡肝、牛肝、鱼、鸡皮（已删除青口贝）
x = cp.Variable(9, nonneg=True)
x_ca = cp.Variable(nonneg=True)   # 碳酸钙补充(mg Ca)
x_tau = cp.Variable(nonneg=True)  # 牛磺酸补充(mg)

R = 3168  # 当前精确配方原料总量

# 营养矩阵（每g食材）
# 蛋白质(g), 脂肪(g), 磷(mg), 钙(mg), 牛磺酸(mg), VA(IU), 铁(mg), 锌(mg), 锰(mg), 碘(mcg)
nutrients = {
    'protein':  np.array([0.196, 0.211, 0.156, 0.177, 0.126, 0.169, 0.204, 0.186, 0.133]),
    'fat':      np.array([0.043, 0.069, 0.093, 0.039, 0.099, 0.048, 0.036, 0.139, 0.324]),
    'phos':     np.array([1.78, 1.97, 1.77, 2.12, 1.98, 2.97, 3.87, 2.17, 0.68]),
    'ca':       np.array([0.09, 0.05, 0.12, 0.07, 0.56, 0.08, 0.05, 0.12, 0.11]),
    'taurine':  np.array([0.40, 0.50, 1.45, 1.60, 0.20, 0.30, 0.40, 0.70, 0.10]),
    'VA':       np.array([0.50, 0.07, 0.31, 0.00, 5.20, 110.78, 168.98, 1.67, 1.50]),
    'iron':     np.array([0.008, 0.008, 0.059, 0.043, 0.018, 0.090, 0.049, 0.0163, 0.007]),
    'mn':       np.array([0.0002, 0.0001, 0.0005, 0.0003, 0.0003, 0.0026, 0.0031, 0.0002, 0.0001]),
}

# 约束
constraints = [
    cp.sum(x) == R,
    
    # 蛋白质 >= 16% 湿基 (按成品2851g)
    nutrients['protein'] @ x >= 0.16 * 2851,
    
    # 钙磷比约束
    (nutrients['ca'] @ x + x_ca) >= 1.1 * (nutrients['phos'] @ x),
    (nutrients['ca'] @ x + x_ca) <= 1.5 * (nutrients['phos'] @ x),
    
    # 牛磺酸（考虑60%烹饪损失）
    nutrients['taurine'] @ x * 0.4 + x_tau >= 2851,  # >= 1000mg/kg
    
    # 肝脏 <= 7%
    x[5] + x[6] <= 0.07 * R,
    
    # 鱼 <= 5%
    x[7] <= 0.05 * R,
    
    # VA上限
    nutrients['VA'] @ x <= 333333 * (2851 * 1.1 / 1000),
    
    # 铁（含补充剂）
    nutrients['iron'] @ x + 36 >= 20 * (2851 * 1.1 / 1000),  # +36mg螯合铁
    
    # 锰（含补充剂）
    nutrients['mn'] @ x + 10 >= 1.2 * (2851 * 1.1 / 1000),  # +10mg Source Naturals
    
    # 各食材比例约束（保持适口性）
    x[0] >= 0.15 * R,  # 鸡腿肉至少15%
    x[0] <= 0.30 * R,
    x[8] >= 0.05 * R,  # 皮脂至少5%
    x[8] <= 0.12 * R,
]

# 目标：接近推荐比例
target = np.array([0.26, 0.19, 0.13, 0.14, 0.08, 0.04, 0.03, 0.04, 0.09]) * R
objective = cp.Minimize(cp.sum_squares(x - target))

prob = cp.Problem(objective, constraints)
prob.solve()

if prob.status == 'optimal':
    names = ['鸡腿肉', '猪瘦肉', '鸡心', '牛心', '鸡蛋', '鸡肝', '牛肝', '鱼', '鸡皮']
    print("=== 优化结果 ===")
    for i, name in enumerate(names):
        print(f"{name}: {x.value[i]:.0f}g ({x.value[i]/R*100:.1f}%)")
    print(f"\n碳酸钙补充钙量: {x_ca.value:.0f}mg → 碳酸钙约 {x_ca.value/400:.1f}g")
    print(f"牛磺酸补充: {x_tau.value:.0f}mg → {x_tau.value/1000:.1f}g")
else:
    print(f"求解失败: {prob.status}")
```
