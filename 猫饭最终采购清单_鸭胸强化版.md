## 四、备料顺序

### 1. 先称重

- [ ] 鸭胸肉去皮后称到 **900g**
- [ ] 鸡腿肉本版为 **0g**，无需准备
- [ ] 猪瘦肉称到 **600g**
- [ ] 鸡心称到 **410g**
- [ ] 牛心称到 **445g**
- [ ] 鸡蛋去壳后总重约 **254g**
- [ ] 鸡肝称到 **127g**
- [ ] 牛肝称到 **95g**
- [ ] 鲅鱼称到 **127g**
- [ ] 鸡皮称到 **222g**

### 2. 再处理

- [ ] 鸭胸肉 **去皮**、切块
- [ ] 猪瘦肉切块
- [ ] 鸡心、牛心洗净切开
- [ ] 鸡蛋煮熟去壳
- [ ] 鸡肝、牛肝切块
- [ ] 鱼处理干净切段
- [ ] 鸡皮切小块
- [ ] **血水保留**

---

## 五、烹饪顺序

- [ ] 大锅加水约 **500ml**
- [ ] 先下：鸭胸肉、猪瘦肉、牛心
- [ ] 刚断生后，下鸡心再煮 **5分钟**
- [ ] 再下鸡肝、牛肝煮 **10分钟**
- [ ] 鱼单独蒸熟
- [ ] 鸡皮单独烤熟或煎熟，**油脂保留**
- [ ] **所有汤汁全部保留**

---

## 六、混合与分装

- [ ] 熟肉 + 蛋 + 鱼 + 鸡皮 + 全部汤汁一起打碎
- [ ] 放凉到 **40°C以下**
- [ ] 按补充剂表全部加入并搅匀
- [ ] 分成 **15份**
- [ ] 每份约 **190g**
- [ ] 当天份放冷藏，其余立刻冷冻

## 八、必须记住的红线

- [ ] 鸭胸必须按 **去皮后重量** 算
- [ ] 不要把鸭皮混进鸭胸位
- [ ] 鲅鱼版建议额外加 **1粒VD3 1000IU**（不加也能29/29达标）
- [ ] 肝脏总量不要超 7%
- [ ] 鱼保持 4%，不要再加高
- [ ] 补充剂必须 **40°C以下** 再加
- [ ] 汤汁和血水不要倒掉
- [ ] 牛磺酸记得 **一半制作时加，一半喂时现加**
